What to watch for:

  • Watch how smaller screens automatically show fewer columns to keep the table readable
  • Use the dropdown menu to select which columns you want to view
  • Notice that some columns, such as Date Issues, are the first to disappear to allow more space for the important columns
  • Note that the 'Name' column can't be hidden by the dropdown because the page design requires it to be shown at all times
