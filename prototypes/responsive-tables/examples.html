Details

Togglable Tables automatically hide and show columns as the viewport allows. Additionally, it provides a menu where the user can explicity toggle any column. The page designer may optionally indicate a level of priority for each column as well as the minimum acceptable width for display.

Developer Notes

Usage:

  • Includes:
    • js/vendor/jquery-ui.widget.js
    • js/app/responsive-tables.js
    • js/foundation/foundation.dropdown.js
  • Optionally use the following classes on <th> elements to prioritize column display:
    • rt-persist — never hides and cannot be toggled
    • rt-p2 — hides when there is not enough space
    • rt-p1 — hides as soon as there is not enough space and all .rt-p2 are already hidden
  • Add the class .not-responsive to a <table> to prevent the plugin from handling it.

ITS conventions:

  • When including a menu, add a .row above the table's .row with an empty div.rt-table-menu to hold the menu.

The plugin is based on Filament Group's responsive tables solution.
