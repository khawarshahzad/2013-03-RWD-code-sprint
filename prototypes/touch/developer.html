Developer Notes

Tap Targets

Add the class .link-buttons-list to a parent container (e.g., div.row) to enable this feature on all child <a> elements.

Note that the links will only turn into buttons if Modernizr.touch is true. It is not dependent on screen size since that's not the reason why you'd want bigger targets.

Touch Library

Instead of binding event listeners to "click", use exlsr.activateEventName. This value will automatically resolve to "tap" (if Modernizr.touch is true) or "click" otherwise. This makes the site much more responsive since mobile devices insert a noticeable delay before dispatching the click event.
