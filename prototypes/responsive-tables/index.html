Responsive Table

See this blog post for a walkthrough of the code and rationale, and the GitHub repo.

Usage:

  • Include:
    • js/vendor/jquery-ui.widget.js
    • js/app/responsive-tables.js
    • js/foundation/foundation.dropdown.js
  • Give the <table> the class .toggle-table
  • Add a div.toggle-table-menu with an id in the form of [table id]-check-container to the page. This will contain the dropdown menu.
  • Optionally use the following classes on <th> elements to prioritize column display:
    • persist (never hides)
    • optional (hides as soon as there is not enough space)
    • essential (only hides when all optional columns are hidden and there is not enough space)
  • Call $('#myTable').toggleTable(); to initialize

ITS conventions:

  • Add a .row above the .row containing the table to hold the div.toggle-table-menu.
