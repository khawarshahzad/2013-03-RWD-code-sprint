Touch

Zepto's touch library modified to work with either Zepto or with jQuery

  • Prototypes Home
  • Touch

Perform any of the following touch actions on this box:

  • tap
  • longTap
  • swipeLeft
  • swipeRight
  • swipeUp
  • swipeDown

The script also supports singleTap, doubleTap, and swipe.

To test on desktop, open Chrome's Dev Tool settings (gear icon) and enable touch events.
